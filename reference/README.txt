Public reference folder
=======================

This directory is intentionally empty in the public repository.

Use it locally for private stage kits, MAME captures, stock-stage analysis,
or generated proof output. The contents are ignored so user-supplied ROM data,
extracted assets, screenshots, nvram/cfg state, and generated BDB/BDD bundles
do not accidentally enter the public repo.

Keep durable, non-asset knowledge in root documentation such as bdd.md,
bbb.md, FEATURES.md, MK2_BACKGROUND_PROOF.md, and MK2_LEVEL_FIDELITY.md.
